2. Rewrite the encryption formula to isolate $m^e$ as itself.

```math
c = (m^e + k)\mod n \\
c ≡ m^e + k \pmod n \\
c - k ≡ m^e \pmod n \\
m^e ≡ (c - k) \mod n
```
